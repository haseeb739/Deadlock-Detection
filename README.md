# Deadlock Detection & Recovery Tool

An interactive browser-based simulator for OS deadlock concepts.

## Features
- Visual Resource Allocation Graph (RAG) with draggable nodes
- DFS-based cycle detection
- Allocation & request matrix
- Recovery strategies: process termination, resource preemption, rollback
- Banker's Algorithm safe sequence check
- 4 preset scenarios (classic, chain, safe, multi-cycle)

## How to Run
Just open `index.html` in any browser. No dependencies, no server needed.

## Concepts Covered
- Resource Allocation Graph
- Deadlock detection via DFS
- Deadlock recovery strategies
- Banker's Algorithm
